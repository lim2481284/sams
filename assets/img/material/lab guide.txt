1. Create a new Laravel project
	- laravel new [projectName] 
	- php artisan serve 



2. Configure application and database
	- localhost/phpmyadmin 
	- create db 
	- change .env file 
		DB_CONNECTION=mysql
		DB_HOST=localhost
		DB_PORT=3306
		DB_DATABASE=test
		DB_USERNAME=root
		DB_PASSWORD=''
	- change config/database.php  : if not working only change this 
	

3. Create a form, for user to key in data
	- create form 
		- resource > view 
			-   <form method='post' action='{{ url('/form') }}'>
			-   <input type="hidden" name="_token" value="{{ session()->token() }}">  : to handle token error 
	
4. Create the respective controller, route and model to save data to database

	- make migration 
		- php artisan make:migration form_table 
			-         Schema::create('form', function (Blueprint $table) {
            $table->increments('id');
            $table->string('content',150);
        });

		- php artisan migrate 

	- create controller
		- php artisan make:controller FormController 
			- public function index()
    {
        return view('form');
    }

	- setup route
		- Route::get('/form', 'FormController@index')->name('home');


	- handle request and make sql call
		- create model 
			- php artisan make:model form 
				-   public $timestamps = false;
  protected $fillable = [
      'content'
  ];


  protected $table = 'form';

		- post request 
			- Route::post('/form', ['as' => '/form', 'uses' => 'FormController@submitForm']);

		- controller 

			-     public function submitForm(Request $request)
    {
        $form = form::create($request->all());
        return redirect()->route('/form');
    }
